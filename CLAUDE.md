# CLAUDE.md — GitHub Profile README

This repo contains a single `README.md` that renders as the GitHub profile page. There is no build system, no dependencies, and no tests.

## Conventions

- **Badges** use Shields.io `img.shields.io/badge/` format with `?logo=<slug>&logoColor=<hex>`. Keep the existing color and logo style when adding new ones. All badges sit in a single block at the bottom of the file, one per line, with no blank lines between them.
- **Mermaid diagrams** use fenced ` ```mermaid ` blocks. Prefer `flowchart TB` with named subgraphs. Keep diagrams simple and readable.
- **Tone** is concise, professional, and technically accurate. No filler.
- **Images** are hosted via GitHub user-attachments and referenced with `<img>` tags inside Markdown tables.

## Git

- SSH remote host alias: `github.com-personal`
- Branch: `main` (single-branch, direct push)
